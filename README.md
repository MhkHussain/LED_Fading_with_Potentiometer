# LED_Fading_with_Potentiometer

## Components:
- Arduino Uno 
- USB cable 
- Breadboard 
- LED x5
- 220 Ohm Resistor x5 (Color Code: Red Red Brown Gold) 
- Jumper wires
- Operating Voltage: 5 volts
